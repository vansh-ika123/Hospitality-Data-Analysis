# Hospitality ETL Data Analytics Project

## Overview
This project demonstrates an end-to-end ETL (Extract, Transform, Load) workflow applied to hospitality booking data, followed by exploratory data analysis to generate business insights.

## ETL Workflow
- Extracted raw hotel booking data from source files
- Transformed data by cleaning, handling missing values, and formatting date columns
- Loaded the processed data for analysis and visualization

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Key Analysis
- Booking trends over time
- Hotel and customer segment performance
- Cancellation and stay pattern analysis

## Notebook
The notebook is shared in an executed, read-only format for easy review.

## Outcome
The analysis helps understand booking behavior and operational trends in the hospitality domain.
